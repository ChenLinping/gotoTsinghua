# 《走向清华》微信小程序 — 发布指南

## 一、注册小程序账号

### 个人账号（推荐学生使用）
1. 访问 https://mp.weixin.qq.com
2. 点击右上角「立即注册」→ 选择「小程序」
3. 使用一个**未注册过微信公众平台**的邮箱完成注册
4. 注册类型选择「个人」
5. 填写身份信息（需要和你一致的真实信息）
6. 完成微信扫码验证

### 注意事项
- 一个身份证最多注册 5 个小程序
- 一个邮箱只能注册一个小程序
- 个人账号**不能**使用微信支付和部分高级接口，但学习类工具小程序完全够用
- 个人账号发布后需要通过审核才能被搜索到

---

## 二、获取 AppID

1. 登录 https://mp.weixin.qq.com
2. 进入小程序后台 → 左侧菜单「开发」→「开发管理」→「开发设置」
3. 复制你的 **AppID（小程序ID）**，格式类似 `wx1234567890abcdef`

---

## 三、配置项目

### 3.1 在微信开发者工具中导入
1. 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)（选择 Windows 64 位）
2. 打开工具 → 选择「导入项目」
3. 项目目录选择解压后的 `gaokao-planner` 文件夹
4. 填入你的 AppID（上面获取的）
5. 点击「确定」

### 3.2 修改 AppID
打开项目中的 `project.config.json`，确保 `appid` 字段是你的真实 AppID：
```json
"appid": "wx你的真实AppID"
```

### 3.3 添加 TabBar 图标（可选）
在项目的 `images` 文件夹中放入 6 个图标文件（建议 81×81 像素 PNG）：

| 文件名 | 用途 |
|---|---|
| tab-home.png | 今日 Tab 默认图标 |
| tab-home-active.png | 今日 Tab 选中图标 |
| tab-book.png | 复习 Tab 默认图标 |
| tab-book-active.png | 复习 Tab 选中图标 |
| tab-chart.png | 统计 Tab 默认图标 |
| tab-chart-active.png | 统计 Tab 选中图标 |

然后在 `app.json` 的 tabBar.list 中为每个页面添加：
```json
{
  "pagePath": "pages/index/index",
  "text": "今日",
  "iconPath": "images/tab-home.png",
  "selectedIconPath": "images/tab-home-active.png"
}
```

> **提示：** 如果暂时不想做图标，不加 iconPath 也可以正常运行，只是 TabBar 只显示文字。

---

## 四、完善小程序信息（发布前必须）

登录 https://mp.weixin.qq.com → 小程序后台：

1. **设置 → 基本设置**
   - 小程序名称：`走向清华`
   - 小程序头像：上传一个正方形图标（建议 256×256）
   - 小程序简介：例如 `高考理科生的一站式学习规划工具，每日任务提醒+六科题库+进度追踪`

2. **设置 → 服务类目**
   - 添加类目：`教育 → 在线教育` 或 `工具 → 信息查询`
   - 个人账号可选的类目有限，「教育」类目通常需要资质，建议选「工具」类目

3. **设置 → 隐私协议**
   - 本小程序不收集用户个人信息，只需填写简单的隐私说明即可

---

## 五、上传代码

1. 在微信开发者工具中确认项目运行正常
2. 点击右上角「上传」按钮
3. 填写版本号（如 `1.0.0`）和项目备注
4. 点击「上传」

上传成功后，代码会出现在小程序后台的「版本管理」中。

---

## 六、提交审核

1. 登录 https://mp.weixin.qq.com
2. 进入「管理」→「版本管理」
3. 在「开发版本」中找到刚上传的版本
4. 点击「提交审核」
5. 填写审核信息：
   - 功能页面：选择首页
   - 功能描述：简要说明小程序的功能
   - 测试账号：如有需要可填写
6. 提交

### 审核时间
- 一般 1-7 个工作日
- 首次提交可能稍慢
- 节假日前后审核可能会延迟

### 审核常见驳回原因
- 名称包含敏感词或与已有小程序重名
- 类目选择不正确
- 小程序功能过于简单（空白页面等）
- 缺少隐私协议

---

## 七、发布上线

审核通过后：
1. 进入「版本管理」
2. 在审核通过的版本旁点击「发布」
3. 确认发布

发布后：
- 用户可通过微信搜索「走向清华」找到你的小程序
- 可以生成小程序码分享给同学

---

## 八、后续更新

每次修改代码后：
1. 开发者工具 → 上传新版本
2. 后台 → 提交审核
3. 审核通过 → 发布

> 小更新可以尝试使用「体验版」让朋友先测试，满意了再提交审核。

---

## 常见问题

**Q：个人账号能用吗？**
可以。工具类小程序个人账号完全可以使用。

**Q：需要花钱吗？**
注册小程序账号免费。如果后续需要认证（企业账号），需要 300 元/年的认证费。个人账号不需要。

**Q：审核被拒怎么办？**
根据驳回原因修改后重新提交即可，不额外收费。

**Q：同学怎么找到我的小程序？**
发布后可通过微信搜索名称找到，也可以生成小程序码/二维码分享。
